# Glossary

- **PII**: Personally Identifiable Information (direct identifiers like name, passport number, etc.).
- **RP / Requester**: Relying party requesting an assurance/claim.
- **Issuer**: Authority issuing base credentials (state provider, licensing body).
- **Predicate proof**: Proof of a property without revealing underlying attribute.
- **Selective disclosure**: Revealing only chosen attributes from a credential.
- **ZKP**: Zero-knowledge proof.
- **AAD**: Associated Auth Data (context binding, anti-swap).
- **AQDR**: Audit Query / Decision Receipt (machine-readable receipt).
- **WORM**: Write Once Read Many (append-only logs).
- **Fail-closed**: default deny on uncertainty.
- **Key Guardian**: abstraction managing keys with explicit protection level.
